# Dataset Description – Facebook Ads Synthetic Dataset

This dataset contains synthetic Facebook Ads performance data for an undergarments brand.

### Columns Meaning

| Column Name      | Meaning |
|------------------|---------|
| date             | The date of the ad performance record |
| campaign_name    | Name of the Facebook Ads campaign |
| adset_name       | Name of the adset under the campaign |
| ad_name          | Creative / ad unit name |
| impressions      | Number of times the ad was shown |
| clicks           | Number of link clicks |
| spend            | Amount spent in INR |
| revenue          | Revenue generated from purchases |
| purchases        | Number of purchases attributed |
| cpc              | Cost per click |
| ctr              | Click-through rate (clicks ÷ impressions) |
| roas             | Return on ad spend (revenue ÷ spend) |
| frequency        | Average number of times a user saw the ad |
| creative_text    | Ad creative’s primary text |
| creative_headline| Ad creative’s headline |
| creative_type    | Format: image, video, carousel, etc. |

### Purpose
This dataset is used to:
- Analyze ROAS drops  
- Identify performance issues  
- Generate insights  
- Validate hypotheses  
- Recommend new creatives
